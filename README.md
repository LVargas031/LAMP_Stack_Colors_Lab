# LAMP_Stack_Colors_Lab
COP4331C LAMP Stack Colors Lab Code

A user can login into the COLORS application, after login the user can search for a color. 

The user has the option to insert a new color name into the database per their account. The user can log out, log back in again, if they search for the new color name that was previously created, it will then pop up to the user in the UI. If the user search for a color name that is not in the database, nothing will happen. If nothing happens, then the color does not exist in the database.
